# Changelog

## 1.0.0 (Initial Production Release)

- **Main-Thread UI & Layout Performance Optimization**: Offloaded all SQLite database queriesŌĆöincluding real-time bookmark status checks (`bookmarkedIds`) via `produceState`ŌĆöto background coroutine dispatchers (`Dispatchers.IO`), ensuring buttery smooth 60fps scrolling and zero main-thread disk I/O during Jetpack Compose layout passes (`onMeasure`).
- **Asynchronous Real-Time News & Snapshot Upserts**: Optimized Cloud Firestore live stream processing (`startLiveArticles`). Incoming news batch updates (`articles` collection) execute local database inserts and upserts asynchronously on IO background threads. Transitioned `moneybytes-apk` database from Datastore Mode to **Firestore in Native mode** (`europe-west2`) for bulletproof out-of-the-box real-time synchronization.
- **Privacy-Friendly Self-Hosted House Ads Engine**: Stripped all third-party tracking and ad SDKs. Transitioned 100% of promotional delivery to privacy-friendly **House Ads** fetched via `HouseAdRepository` (`https://house.monkey-network.xyz/api/public/ads`). Banners dynamically intersperse into the news stream (`NewsList`) after every 4th story (`itemCount % 4 == 0`) and embed directly on article summary pages (`StoryDetail`) with clean `"Promo"` and `"Tap to learn more"` badges.
- **Offline Bookmarks & "Saved Stories" Section**: Added dedicated **Saved Stories (`SAVED`)** section in the main navigation menu. Built on `BudgieArticleDatabase` SQLite storage (`bookmarked_articles`), allowing users to bookmark and read stories anywhere, even offline, with non-blocking tactile actions (`onToggleBookmark` via `scope.launch(Dispatchers.IO)`).
- **Dedicated "General News" (`GENERAL`) Category**: Added a new main section tab (`General`) across all 4 outlets (BBC, Sky, Guardian, Sun), intelligently separating everyday general reporting, lifestyle, and culture news from urgent breaking news alerts.
- **Hybrid Neural Cloud & On-Device TTS Audio Reader (`<1.2s` Playback)**: Built a hybrid audio streaming engine (`BudgieAudioReader`). When online, it connects to an unauthenticated public voice API streaming the ultra-natural `Brian` (British English Male News Anchor) neural voice (`1.0x -> 1.25x -> 1.5x`). If network voice preparation exceeds 1.2s or when offline, it instantly switches to high-quality on-device neural `TextToSpeech` with sentence-by-sentence queueing (`QUEUE_ADD`) and explicit lifecycle/binding awareness (`isTtsReady`, `pendingTtsAction`).
- **Instant Keyword Search**: Added an interactive search toggle in the Top Bar (`Icons.Rounded.Search`). Users can instantly filter loaded news stories by keyword or topic matching titles, descriptions, and source outlets.
